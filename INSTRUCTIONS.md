# 🚀 ИНСТРУКЦИЯ ПО ЗАПУСКУ

## Быстрый старт (3 минуты)

### Шаг 1: Установите зависимости
```bash
pip install -r requirements.txt
```

### Шаг 2: Создайте .env файл
Создайте файл `.env` в корне проекта:
```env
BOT_TOKEN=ваш_токен_от_BotFather
SECRET_KEY=любая_длинная_строка_для_jwt
```

**Как получить BOT_TOKEN:**
1. Откройте Telegram
2. Найдите @BotFather
3. Отправьте: `/newbot`
4. Придумайте имя бота
5. Скопируйте токен и вставьте в .env

### Шаг 3: Запустите проект

**Вариант A - Автоматический (Linux/Mac):**
```bash
./start.sh
```
Выберите вариант 3 (оба сервиса)

**Вариант B - Вручную:**

**Терминал 1 - Telegram бот:**
```bash
cd backend
python bot.py
```

**Терминал 2 - Web-сервер:**
```bash
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

**Вариант C - Docker:**
```bash
docker-compose up
```

### Шаг 4: Откройте проект

- 🌐 **Веб-интерфейс:** http://localhost:8000
- 📱 **Telegram-бот:** Найдите вашего бота в Telegram и напишите `/start`

## 🐛 Решение проблем

### Бот не отвечает:
- Проверьте BOT_TOKEN в .env
- Убедитесь что bot.py запущен
- Проверьте интернет-соединение

### Сайт не открывается:
- Проверьте что main.py запущен
- Попробуйте другой порт: `--port 8080`
- Очистите кеш браузера

### Ошибка "ModuleNotFoundError":
```bash
pip install -r requirements.txt
```

### База данных не создаётся:
- Удалите tasks.db
- Перезапустите проект
- БД создастся автоматически

```bash

alembic init alembic

# Создание миграции
alembic revision --autogenerate -m "описание"

alembic upgrade head
```
